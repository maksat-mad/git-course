## Қазіргі commit туралы ақпарат алу

```git
git show --pretty=fuller
```

немесе

```git
git show <identifier>
```

<br/>
<br/>

## Авторды осылай өзгертсе болады

```git
git commit --author='Maksat Mad <maksat@gmail.com>' --date='...'
```

<br/>
<br/>

## Кез келгенін өзгертсе болады

```git
GIT_AUTHOR_NAME
GIT_AUTHOR_EMAIL
GIT_AUTHOR_DATE
GIT_COMMITTER_NAME
GIT_COMMITTER_EMAIL
GIT_COMMITTER_DATE
```
